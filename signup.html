Integration Path

Direct sync is possible, but it needs a helper service.

Because GitHub Pages is a static host, a live SignupGenius-to-website sync usually needs an outside step such as a Google Sheet, GitHub Action, or small serverless function. That is very doable later, but it is not something GitHub Pages can safely do by itself.

Recommended First Step

Start simple and professional

Use this page as your official signup portal now. Once your workflow stabilizes, we can add a sync layer that publishes confirmed signups into a table or calendar feed.

Future Automation

One clean path to syncing signup data.

This is the most realistic architecture if you want SignupGenius participation to appear automatically on a GitHub Pages site.

1. SignupGenius

Members claim slots through the signup page you already use.

2. Sheet or Export

Pull signup data into a Google Sheet or exported CSV.

3. Automation

Use a script, GitHub Action, or Apps Script to convert the data.

4. JSON Output

Publish a clean data file the website can read safely.

5. Site Update

Show live staffing counts or names on the calendar and roster pages.
